Project Description

A Full stack E-commerce book store website. This website includes a user authentication system for login, and registration page for first-time users to register. Users are able to add to their cart and process their payment. Their purchases will be saved in the database for future retrievals. There is an admin feature which allows the admin to add, delete, or modify the inventory through a user interface.
