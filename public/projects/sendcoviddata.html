COVID-19 Messenger

Project Description

Created a web scraper using Python and BeautifulSoup to extract COVID-19 data of all 50 states. I then decided to integrate that into a web application. This tool will send you real-time data on all 50 states in the United States in table format. Data includes total cases, new cases, total deaths, and new deaths.
